Lab 18: Libraries and Frameworks 1

Challenges

Lab 18 was our intro to libraries and Frameworks and we were tasked to use bootstrap to add dependencies, create a nav bar and alter our layout using their libraries.

Problems

The problems that I ran into was trying to use the bootstrap grid layout, using bootstrap was a weakness of mine in 251 so for this lab I worked along side some friends in addition to my partner.

Results
